# Design Trends - 最新デザイントレンド調査

最新のUIデザイントレンドを調査し、要約レポートを生成します。

## 使い方

```
/design-trends [カテゴリ]
```

- `$ARGUMENTS` - オプションのカテゴリフィルター

---

## タスク

### 1. トレンドリサーチ

以下のソースから最新デザイントレンドを調査：

1. **Dribbble** - プロフェッショナルデザイナーの作品
2. **Awwwards** - 受賞Webサイト
3. **Mobbin** - モバイルUIパターン

### 2. 分析項目

- **スタイル傾向**: Minimalist, Glassmorphism, Bento Grid, Dark Mode等
- **カラートレンド**: 人気のカラーパレットと配色
- **タイポグラフィ**: フォント選択とサイズ傾向
- **レイアウト**: グリッドシステムとコンポーネント配置
- **インタラクション**: アニメーションとマイクロインタラクション

### 3. 出力形式

```markdown
## 2024 デザイントレンドレポート

### トップトレンド
1. **Bento Grid** (78% popularity)
   - 特徴: グリッドベースのカード配置
   - 使用例: Apple, Spotify

2. **Glassmorphism** (65% popularity)
   - 特徴: 透明感とブラー効果
   - 使用例: iOS, Windows 11

### カラートレンド
- プライマリ: Deep Purple (#6366F1)
- アクセント: Vibrant Teal (#14B8A6)

### 推奨アクション
- [具体的な実装提案]
```

## 実行

デザインスカウトエージェントを使用してトレンドを分析し、結果を報告してください。
